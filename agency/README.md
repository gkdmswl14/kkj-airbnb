# 호텔 웹 에이전시

호텔 예약 및 관리 시스템입니다.

## 주요 기능

### 사용자 기능
- ✅ 호텔 검색 및 필터링
- ✅ 호텔 상세 정보 조회
- ✅ 예약 시스템
- ✅ 예약 내역 확인

### 관리자 기능
- ✅ 호텔 등록 및 관리
- ✅ 예약 관리
- ✅ 통계 대시보드

## 파일 구조

```
agency/
├── index.html          # 메인 페이지
├── hotels.html         # 호텔 목록 페이지
├── hotel-detail.html   # 호텔 상세 페이지
├── bookings.html       # 예약 내역 페이지
├── admin.html          # 관리자 대시보드
├── css/
│   └── agency.css      # 스타일시트
└── js/
    ├── app.js          # 메인 애플리케이션 로직
    ├── hotels.js       # 호텔 목록 기능
    ├── hotel-detail.js # 호텔 상세 기능
    ├── admin.js        # 관리자 기능
    └── bookings.js     # 예약 내역 기능
```

## 사용 방법

### 1. 로컬 서버 실행

```bash
# Python 사용
cd agency
python -m http.server 8080

# 또는 프로젝트 루트에서
cd ..
python -m http.server 8080
```

그 다음 브라우저에서 `http://localhost:8080/agency/index.html` 접속

### 2. 주요 페이지

- **메인 페이지**: `index.html`
- **호텔 목록**: `hotels.html`
- **호텔 상세**: `hotel-detail.html?id=1`
- **예약 내역**: `bookings.html`
- **관리자**: `admin.html`

## 데이터 저장

모든 데이터는 브라우저의 LocalStorage에 저장됩니다:
- 호텔 정보
- 예약 정보
- 설정 정보

## 기본 호텔 데이터

시스템에는 6개의 샘플 호텔이 포함되어 있습니다:
1. 그랜드 호텔 서울
2. 제주 리조트
3. 부산 해운대 호텔
4. 경주 힐튼 호텔
5. 강릉 오션뷰 호텔
6. 서울 명동 호텔

## 기능 상세

### 검색 및 필터
- 도시/호텔명 검색
- 가격 범위 필터
- 별점 필터
- 편의시설 필터
- 정렬 기능 (이름, 가격, 평점)

### 예약 시스템
- 체크인/체크아웃 날짜 선택
- 인원 선택
- 예약자 정보 입력
- 자동 가격 계산
- 예약 확인

### 관리자 대시보드
- 호텔 추가/삭제
- 예약 관리
- 통계 조회 (총 호텔 수, 예약 수, 매출, 사용자 수)
