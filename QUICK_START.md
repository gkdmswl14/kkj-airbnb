# 🚀 빠른 시작 가이드

## ⚠️ 중요: Git 설치 필요

GitHub에 연결하려면 **Git이 반드시 필요**합니다.

## 1단계: Git 설치 (약 5분)

방금 브라우저에서 Git 다운로드 페이지가 열렸습니다. 

**설치 방법:**
1. "Download for Windows" 버튼 클릭
2. 다운로드된 `.exe` 파일 실행
3. 설치 중 **모든 기본 옵션을 그대로 사용** (Next만 클릭)
4. 설치 완료 후 **이 터미널을 완전히 종료하고 다시 열기**

## 2단계: Git 설치 확인

터미널을 다시 연 후:
```powershell
git --version
```

`git version 2.x.x` 같은 메시지가 나오면 성공!

## 3단계: GitHub 연결

다음 명령어 실행:
```powershell
.\setup-github.ps1
```

또는:
```cmd
setup-github.bat
```
