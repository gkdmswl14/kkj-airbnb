# 404 오류 해결 방법

## 문제
GitHub에서 저장소를 찾을 수 없습니다 (404 NOT_FOUND)

## 해결 단계

### 방법 1: GitHub에서 저장소 수동 생성 (권장)

1. **저장소 생성 페이지 열기**
   - https://github.com/new 접속
   - 방금 브라우저에서 열었습니다

2. **저장소 정보 입력**
   - Repository name: `kkj-airbnb`
   - Description: (선택사항) "Colina Hotel Template"
   - Public 또는 Private 선택
   - **"Initialize this repository with" 체크박스는 모두 해제** (이미 파일이 있으므로)
   - "Create repository" 클릭

3. **로컬 저장소와 연결**
   ```powershell
   # 원격 저장소 제거
   git remote remove origin
   
   # 새로 추가
   git remote add origin https://github.com/gkdmswl14/kkj-airbnb.git
   
   # 푸시
   git push -u origin main
   ```

### 방법 2: 저장소가 이미 있다면

저장소가 이미 존재한다면, 비공개로 설정되어 있을 수 있습니다.

1. https://github.com/gkdmswl14/kkj-airbnb 접속
2. Settings > Danger Zone > Change visibility
3. "Make public" 선택

### 방법 3: 토큰 권한 확인

토큰에 저장소 접근 권한이 있는지 확인:

1. https://github.com/settings/tokens 접속
2. 토큰에 `repo` 권한이 있는지 확인
3. 없다면 새 토큰 생성
